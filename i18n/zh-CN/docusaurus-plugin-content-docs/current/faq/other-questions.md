# 其他问题

## 8.1. 行为修改功能
LiveContainer 的主要目标是按照App被自签的方式运行App，而不改变其预期功能。因此，不会考虑修改App行为的功能请求（例如网络监控、禁用功能）。但是，欢迎社区为这些类型的功能做出贡献。

## 8.2. App因需要仅可用于付费开发者账户的权限而崩溃/不能正常工作
LiveContainer 的设计目的是在某种程度上绕过免费开发者账户上设置的 3 个App限制。由于需要仅可用于付费开发者账户的权限（例如推送通知、Siri、iCloud）而导致的App崩溃/不能正常工作将不会被修复。

## 8.3. UI 美化/自定义功能
对于诸如主页自定义、更改App名称/图标和其他视觉调整的功能的请求优先级较低，因为它们不太有用，可能会减速App启动时间。您可以简单地在主屏幕上创建快捷方式，并改变它们的顺序和它们的名称/图标。


## 8.4. 关于游戏
游戏有关的问题修复难度较大，找到具体原因十分困难，请放低期待。

此外, 我们不提供对游戏作弊、破解、作弊客户端的支持。有关游戏作弊的issue会被立刻关闭。请使用游戏的官方客户端，或使用其他自签方式。
