# 使用插件

LiveContainer 支持在不修改 IPA 的情况下注入插件。由于并非所有插件都能与 TweakLoader 完全兼容，若条件允许依旧建议直接对应用进行注入。

LiveContainer 自带的 TweakLoader 会自动加载 CydiaSubstrate 与插件，并在你安装的每个应用中注入。如有需要，可以替换 `TweakLoader.dylib` 符号链接来自定义实现。

`Tweaks` 目录下的 `.dylib` 默认对所有应用生效。你也可以创建特定应用的插件文件夹，并在不同文件夹之间快速切换。

安装插件时，可以使用内置的插件管理器，它会在导入时自动为插件签名。也可以手动拷贝文件，再通过管理器进行签名。

## 添加插件

:::note
插件管理仅限主 LiveContainer（蓝色）以及私有应用使用。如果想在共享应用中启用插件，请先将应用转换为私有，添加插件后再改回共享，插件依旧有效。
:::

1. 进入插件标签页，点击加号：
   - 若要对所有应用注入（全局插件），点击「导入插件」
   - 若只想给某个应用注入，点击「新建文件夹」
2. 全局插件导入完成后即可直接启动应用验证
3. 针对特定应用：进入刚创建的文件夹，点击加号 → 导入插件
4. 针对特定应用：长按应用 → 设置 → 插件文件夹 → 选择你创建的文件夹

## 删除插件

进入插件标签页，找到插件/文件夹，向左滑动后删除。

## 重命名插件

在插件标签页长按插件/文件夹，然后选择重命名。

## 关于插件签名

LiveContainer 会自动处理插件签名：当插件有更改或签名过期时会自动重新签名。你也可以点击加号旁边的签名按钮手动触发。
