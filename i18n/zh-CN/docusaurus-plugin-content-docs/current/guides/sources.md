# 源

`源` 选项卡允许您浏览 AltStore 风格的源并安装应用。

## 添加源

1. 点击（`+`）添加源。
2. 输入源 URL。
3. 确认添加。

## 深度链接支持

可以通过链接添加源：

- `livecontainer://source?url=<encoded_source_url>`

收到后，LiveContainer 切换到 `源` 选项卡并尝试添加该源 URL。

## 移除或刷新源

- 在源管理中使用源菜单（`...`）或滑动操作。
- `刷新` 重新加载一个源。
- `全部刷新` 重新加载所有已添加的源。

## 从源安装应用

1. 展开源部分。
2. 找到应用并点击 `安装`。
3. LiveContainer 切换到 `App` 选项卡并使用应用的最新版本下载 URL 开始安装。

:::note
如果源包含超过 50 个应用，由于性能问题，您需要使用搜索按钮查找应用。
:::

## 常见问题

- **Invalid URL**: 源输入格式错误。
- **Duplicate source**: 此确切 URL 已添加。
- **Unable to refresh source**: 网络/服务器/JSON 格式问题。
- **No apps available**: 源已解析但应用列表为空。
