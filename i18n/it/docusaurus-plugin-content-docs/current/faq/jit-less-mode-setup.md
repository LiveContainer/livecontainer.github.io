# 2. Configurazione Modalità JIT-Less

**Leggi le istruzioni sulla configurazione della modalità JIT-Less prima di sottomettere issue su questo.**

## 2.1. Cannot Access App Group

C'è qualcosa che non va col setup di SideStore/AltStore e LiveContainer. Vedi la [Sezione 6](https://github.com/LiveContainer/LiveContainer/issues/265#issuecomment-2558409380).

## 2.1. SideStore crasha dopo avergli applicato la patch

Segui la guida per l'installazione di SideStore prima di applicare la patch. es. esegui il sideload di SideStore ipa in SideStore immediatamente dopo aver sideloadato SideStore da AltServer.

## 2.2. (SparseStore) Aperto SideStore sbagliato / SideStore non è più disponibile

1. Patcha SideStore -> Archive Only
2. SideStore patchato è salvato nella cartella documenti di LiveContainer, puoi trovarla nell'app "File". Fai il sideload nel SideStore giusto per applicare la patch.
