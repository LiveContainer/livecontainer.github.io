# Introduzione a LiveContainer

Un'app launcher che esegue le app iOS senza installarle davvero!

- LiveContainer è un'app launcher (non un emulatore o hypervisor) che ti permette di eseguire app iOS al suo interno.
- Ti consente di installare infinite app (il limite di 3 app/ 10 app id dell'account developer gratuito non conta!) e installare diverse versioni di una stessa app con diversi container di dati.
- Quando è disponibile JIT, codesign è completamente bypassato, non c'è bisogno di firmare le app prima di installarle. In caso contrario le app verranno firmate con lo stesso certificato di LiveContainer.

## Feature

- Esegui app iOS in container isolati
- Jailbreak non necessario
- Ambiente sandbox sicuro
- Gestione app semplice
- Supporto per app sideloaded
