# LiveContainer Multipli

Installando più LiveContainer e convertendo le app ad App Condivise, puoi aprire un'app tra diversi LiveContainer mantenendo la maggior parte dei sui dati e impostazioni.

## Installazione di LiveContainer Multipli

Per usare più LiveContainer:

1. Nelle impostazioni tocca "Installa un altro LiveContainer"
2. Il primo container (icona blu) si esegue di default
3. Per usare il secondo container (icona grigia):
   - Converti un'app ad "App condivisa" nel primo container
   - Esegui l'app dal secondo container
