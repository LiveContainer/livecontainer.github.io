# Blocca e Nascondi App

LiveContainer offre tre livelli per nascondere le app:

* Blocca app: Nelle impostazioni per app puoi bloccare un'app, facendo sì che ti venga richiesto il FaceID o una password per eseguirla.
* Blocca app nascoste: Una volta bloccata, puoi nascondere un'app. Verrà mostrata come placeholder nella sezione "App Nascoste" e dovrai autenticarti per visualizzarla o eseguirla.
* Strict Hiding Mode: Per nasconderla completamente abilita "Strict Hiding Mode" nelle impostazioni di LiveContainer dopo esserti autenticato. Le app nascoste non verranno più mostrate nell'interfaccia. Per visualizzarle, premi tre volte il testo "x App in totale" nella schermata principale e autenticati per sbloccare le app quando esegui LiveContainer di nuovo.
